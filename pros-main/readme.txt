1.download a mingw64 and save it on disk C
2.add mingw64 to environment path (for example if u have C:\mingw64), add to path C:\mingw64 and C:\mingw64\bin
3.check if your mingw64(compilator for example gcc) works by writting in comand Prompt: gcc -version
4.If u have it installed you can turn on a program by go to the folder with this program and write: gcc filename.c -o output
5.In this folder should show up a "output.exe" (because you gave "output" name in 4-th step to compile as)
6.Click output.exe and your program works
